Какие файлы имеются:

1. docs_find_all_embeddings.py — скрипт для создания эмбеддингов. 
	Создание эмбенддингов из .docx файла. С последующим сохранением в БД Milvus (текст, таблицы, названия изображения).
	Также сохранение всех таблиц и изображений в БД MiniO

2. TG_and_GPT.py - скрипт для запуска Бота.
Является посредником между ТГ ботом и GPT. 
Конвертирует из запроса юзера эмбеддинг, ищет наиболее похожие эмбеддинги в БД Milvus. Отправляет эти эмбеддинги в качестве контекста GPT, после чего GTP отправляет готовый ответ ТГ Боту.

----------------------------------------------
Как пользоваться:

Шаг 1. 
Подготовить данные создания по ним эмбеддингов с последующим поиском. Переместить их по нужному пути (путь можно посмотреть/поменять в файле docs_find_all_embeddings.py).

Шаг 2.
Включить VPN, запустить скрипт docs_find_all_embeddings.py.

Шаг 4. 
Включить VPN, запустить скрипт TG_and_GPT.py

Шаг 5. 
Написать ТГ боту
