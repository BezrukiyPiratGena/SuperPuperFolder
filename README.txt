Какие файлы имеются:

1. pdf_to_txt.py - скрипт для конвертирования .pdf в .txt.
	Конвертация файл(pdf) в .txt под именем extracted_text.txt

2. txt_to_embeddings.py — скрипт для создания эмбеддингов. 
	Создание эмбенддингов из .txt файла. С последующим сохранением в БД Milvus (сопоставляя текст с его эмбеддингам).

3. TG_and_GPT.py - скрипт для запуска Асистента. 
Является посредником между ТГ ботом и GPT. 
Конвертирует из запроса юзера эмбеддинг, ищем наиболее похожие эмбеддинги в БД Milvus. Отправляет эти эмбеддинги в качестве контекста GPT, после чего GTP отправляет готовый ответ ТГ Боту.

4. tokens.env - файл с токенами OpenAI и TelegramBot

----------------------------------------------
Как пользоваться:

Шаг 1. 
Подготовить данные создания по ним эмбеддингов с последующим поиском. Переместить их по нужному пути (путь можно посмотреть/поменять в файле pdf_to_txt.py).

Шаг 2.
Запустить скрипт pdf_to_txt.py. После чего появится файл extracted_text.txt (путь можно посмотреть/поменять в файле pdf_to_txt.py)

Шаг 3.
Запустить скрипт txt_to_embeddings.py. 

Шаг 4. 
Включить VPN, Запустить скрипт TG_and_GPT.py

Шаг 5. 
Написать ТГ боту
